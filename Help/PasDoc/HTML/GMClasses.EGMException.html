Class EGMException

Unit

GMClasses

Declaration

type EGMException = class(Exception)

Description

Base class for all Exception on GMLib.

If you want to translate the exceptions, follow this steps:

  1. Create a new element into GMSets.TLang set representing the new language.

  2. Create a new const array GMTranslations unit named Lang_XXX.

  3. And put new set value into GMTranslations.GetTranslateText case.
