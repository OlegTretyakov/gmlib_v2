TGMControlPosition = (...);

Identifiers used to specify the placement of controls on the map.

Controls are positioned relative to other controls in the same layout position. Controls that are added first are positioned closer to the edge of the map. 

  +----------------+
 + TL    TC    TR +
 + LT          RT +
 +                +
 + LC          RC +
 +                +
 + LB          RB +
 + BL    BC    BR +
 +----------------+

Elements in the top or bottom row flow towards the middle of the row. Elements in the left or right column flow towards the middle of the column.

More info at https://developers.google.com/maps/documentation/javascript/reference#ControlPosition

TGMZoomControlStyle = (...);

google.maps.ZoomControlStyle constants

Styles for the zoom control.

Note: Zoom control styles are not available in the new set of controls introduced in v3.22 of the Google Maps JavaScript API. While using v3.22 and v3.23, you can choose to use the earlier set of controls rather than the new controls, thus making the large Zoom control available as part of the old control set. See What's New in the v3.22 Map Controls at https://developers.google.com/maps/articles/v322-controls-diff.

More info at https://developers.google.com/maps/documentation/javascript/reference#ZoomControlStyle
