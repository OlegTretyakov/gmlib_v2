Declaration

type TGMMapTypeStyle = class(TGMCustomMapTypeStyle)

Description

google.maps.MapTypeStyle object from Google Maps API.

The TGMMapTypeStyle is an individual object of collection of selectors and stylers that define how the map should be styled. Selectors specify what map elements should be affected and stylers specify how those elements should be modified.

More information at https://developers.google.com/maps/documentation/javascript/reference#MapTypeStyle
